Tipos de programacion web

Front - End

Front - End
Caracteristicas

  1. La programación es del lado del cliente

  2. Tiene interacción directa con el usuario

Tecnologías base del Front - End

Back - End

Back - End
Caracteristicas

  1. La programación es del lado del servidor

  2. Se da la logia del negocio

  3. Se ve la seguridad y el rendimiento del sitio web

  4. No es estrictamente Web

Tecnologías base del Front - End

API

API
Caracteristicas

  1. Conecta el Front - Ent con el Back – End

  2. El Back - End controla los datos en APIs de tipo REST

  3. El Front - End controla los datos en APIs de tipo GraphQL
